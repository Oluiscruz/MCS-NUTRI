# NutriMS

Português (PT-BR)

Descrição
- NutriMS é um sistema de gestão para consultas e perfis de nutricionistas e pacientes. O projeto contém um frontend em React (Vite) e um backend em Node.js com rotas de autenticação e persistência em banco SQL (arquivo `sistema_nutricao.sql`).

Linguagens e frameworks utilizadas
- Linguagens: JavaScript, HTML, CSS/SCSS, SQL
- Frontend: React, Vite, Sass (SCSS)
- Backend: Node.js, Express
- Autenticação: OAuth/Google (integrado no frontend via `GoogleCallback.jsx`)

Principais pastas
- `src/`: código-fonte do frontend (páginas, componentes, estilos)
- `public/`: ativos públicos
- `backend/`: servidor Node.js, configuração de BD e APIs

Observações
- O arquivo `sistema_nutricao.sql` contém o esquema/seed do banco de dados para uso local.
- Arquivos como `eslint.config.js` e `vite.config.js` mostram que o projeto usa ESLint e Vite para desenvolvimento.

English (EN)

Description
- NutriMS is a management system for appointments and profiles of nutritionists and patients. The project includes a React (Vite) frontend and a Node.js backend with authentication routes and a SQL database schema (file `sistema_nutricao.sql`).

Languages and frameworks used
- Languages: JavaScript, HTML, CSS/SCSS, SQL
- Frontend: React, Vite, Sass (SCSS)
- Backend: Node.js, Express
- Authentication: OAuth/Google (frontend integration via `GoogleCallback.jsx`)

Main folders
- `src/`: frontend source code (pages, components, styles)
- `public/`: public assets
- `backend/`: Node.js server, DB config and APIs

Notes
- The `sistema_nutricao.sql` file contains the database schema/seed for local setup.
- Files such as `eslint.config.js` and `vite.config.js` indicate ESLint and Vite are used for development.
